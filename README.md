# TDSoulLike
 TopDownSoulLike using GAS


# Inputs

- move 		        : Mouse Right Click
- GetTargetInfo     : Mouse Left Click
- Sprint            : Left Shift
- Dash             	: Space
- Default Atk     	: A
- Smash Atk      	: S
- Special Skill	    : D
- Block		        : F
- Interaction	    : E
- Change Posture	: Q
- LockDown Target   : W
